Sayfa üzerinde bir takım verileri tablo formatında göstermek için size sunulan element "table" elementidir. "tr", "td", "th"

TD(table-data) => Kolon bilgisini, TH(table-header) => Başlık bilgisini, TR(table-row) => Satır biligisini temsil eder.

Bazı öznitelik:

  • Width => Tablonun genişliğini verir
  • Border => Tablonun iç ve dış çerçeveleri için bir boyut verilir.
  • Cellspacing => Hücrelerin birbirine olan uzaklıklarını verir.
  • Cellpadding => Hücre içerisindek, değerlerin dış çerçeveye olan uzaklıklarını verir.
  • Colspan => Hücrelerin merge edilmesini(birleştirilmesini) sağlayan özniteliktir. Aşağıdaki tabloda vermiş olduğumuz 2 değeri o hücrenin 2 sütunlukmuş gibi favranmasını söyler.
  • Rowspan => Satırların merge edilmesini sağlar
